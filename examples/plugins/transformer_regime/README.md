# Transformer Regime Detector Plugin (CSV)

This plugin reads regime probabilities from a CSV generated by the Python transformer example.

## Build

```bash
cmake -S examples/plugins/transformer_regime -B examples/plugins/transformer_regime/build \
  -DREGIMEFLOW_ROOT=$(pwd) -DREGIMEFLOW_BUILD=$(pwd)/build
cmake --build examples/plugins/transformer_regime/build
```

If libtorch is available, a TorchScript detector target will also be built:

- `libtransformer_torchscript_detector.so`

## Config

```yaml
plugins:
  search_paths:
    - examples/plugins/transformer_regime/build
  load:
    - libtransformer_regime_detector.so

regime:
  detector: transformer_regime
  params:
    signals_path: examples/python_transformer_regime/regime_signals.csv
```

TorchScript config (requires libtorch):

```yaml
plugins:
  search_paths:
    - examples/plugins/transformer_regime/build
  load:
    - libtransformer_torchscript_detector.so

regime:
  detector: transformer_torchscript
  params:
    model_path: /path/to/regime_transformer.pt
    feature_window: 120
    feature_dim: 8
```
